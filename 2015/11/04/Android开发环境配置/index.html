Windows下Android开发环境配置

Nov 04 2015

Java SE Development Kit

  1. 去Oracle官网下载JDK；
  2. 建议安装JDK到C盘，安装到其他盘可能会有问题；
  3. 安装完成后配置环境变量
    1） 打开 计算机->属性->高级系统设置->环境变量->系统变量；
    2） 新建系统变量，变量名JAVA_HOME，变量值填JDK安装路径，例：C:\Program Files\Java\jdk1.8.0_65；
    3） 新建系统变量，变量名classpath，变量值%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar；
    4） 编辑变量Path，添加%JAVA_HOME%\bin，主要以英文分号隔开。
    5） 打开命令行，输入java -version检查是否配置成功

Android Studio

  1. 去Android Tools Project Site下载Android Studio，需科学上网；
  2. 页面上有Canary，Dev，Beta和Stable 4个渠道，Stable的是稳定版，点开Stable然后选择最新版；
  3. Windows下有3个可选项：IDE+SDK，IDE，Plain，前两个是安装版，区别在于是否捆绑了SDK，后一个是免安装版，解压出来就可以用，不包含SDK，launcher在bin目录下，按需下载。

Gradle

  1. 去Gradle官网下载Gradle；
  2. 解压Gradle压缩包，放到自己能找到的地方，可以和Android Studio放在一起；
  3. 打开Android Studio，File->Settings，搜索Gradle，选中Use local gradle distribution，选择Gradle home，例：D:/Program Files/Android/gradle/gradle-2.8
  4. 用自己的Gradle的一个好处是省去了漫长的下载过程。
