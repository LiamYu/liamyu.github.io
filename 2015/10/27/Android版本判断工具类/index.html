Android版本判断工具类

Oct 27 2015

简易的版本判断工具，包括了几个常用版本的判断：ICS、JellyBean、KitKat、Lollipop和Marshmallow，写法如下： 

/**
 * ICS API level 14
 *
 * @return
 */
public static boolean hasICS() {
    return Build.VERSION.SDK_INT >= Build.VERSION_CODES.ICE_CREAM_SANDWICH;
}
